# Agents

> [!IMPORTANT]
> 이 파일은 AI 코딩 에이전트를 위한 지침(README for Agents)입니다. 에이전트는 작업을 시작하기 전 이 문서의 지침을 반드시 숙지해야 합니다.

## Agent Instructions

이 프로젝트에서 활동하는 모든 에이전트는 다음 규칙을 엄격히 준수합니다.

### 1. 언어 및 커뮤니케이션
- 모든 답변, 기술 문서, 코드 주석은 **한국어**로 작성합니다.
- 전문 용어는 영어 원문을 병행 표기할 수 있으나 문장의 주 언어는 한국어여야 합니다.

### 2. 작업 프로세스
- **계획 우선**: 복잡한 작업을 수행하기 전 `implementation_plan`을 작성하여 공유합니다.
- **문서 동기화**: 프로젝트 코드 변경 시 관련 문서(`AGENTS.md` 등)를 즉시 업데이트합니다. (단, **`README.md`는 사용자의 명시적 요청이 없는 한 업데이트하지 않습니다.**)
- **검증**: 작업 완료 후 테스트나 명령어를 통해 결과를 검증하고 증거를 제시합니다.

### 3. 기술 활용 (Stack & Capabilities)
- **MCP**: 프로젝트 내 제공되는 MCP 서버를 적극적으로 활용합니다.
- **Skills**: `.agents/skills` 또는 시스템의 글로벌 스킬을 활용하여 워크플로우를 자동화합니다.

### 4. 에이전트 자산 관리 (.agents 디렉토리)
에이전트 관련 자산은 다음 구조로 관리하며, 필요한 경우 적극적으로 참조 및 활용합니다.
- **`.agents/rules/`**: 프로젝트 특화 가이드라인 및 규칙 저장
- **`.agents/workflows/`**: 복잡한 작업 자동화를 위한 단계별 워크플로우 저장
- **`.agents/skills/`**: 특정 기술 스택이나 기능을 위한 에이전트 능력(Skills) 저장
