# Github Desktop
[Github Desktop](https://desktop.github.com/)은 깃허브에서 개발한 깃 GUI 클라이언트 도구이다. Git 리모트 저장소로 Github을 사용하는 경우 Github Desktop으로 계정과 쉽게 연동할 수 있다.
