# Update Git Version
윈도우에 설치되는 Git은 별도의 업데이트 명령어를 제공한다.

## Windows
```powershell
git update-git-for-windows
Git for Windows 2.32.0.windows.2 (64-bit)
Up to date
```

## MacOS
```zsh
brew update git
```

## Ubuntu
```bash
sudo add-apt-repository ppa:git-core/ppa -y
sudo apt-get update
sudo apt-get install git -y
```
