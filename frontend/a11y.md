# Frontend > Accessibility

웹 접근성 또는 디지털 접근성으로 장애 여부와 상관없이 동등한 서비스를 제공하는 것을 목표로 함

- 시맨틱 HTML
- 대체 텍스트 지원
- 명확한 색상 및 대비
- 키보드 및 스크린 리더 사용자 지원
- [WCAG 2.2](https://www.w3.org/TR/WCAG22/)

## KWCAG

한국형 웹 콘텐츠 접근성 지침(KWCAG)은 한국웹접근성평가원에서 국제적 표준으로 정의한 [WCAG](https://www.w3.org/WAI/standards-guidelines/wcag/) 권장사항을 기반으로 만든 심사 기준으로 [대한민국 정부 디자인 시스템(KRDS)](https://www.krds.go.kr/html/site/utility/utility_04.html)에서도 웹 접근성을 준수하는 것을 목표로 함
